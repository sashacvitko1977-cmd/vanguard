VANGUARD — как запустить сайт
================================

СПОСОБ 1 (самый простой)
------------------------
Дважды кликните по файлу:

  OPEN.bat

Сайт откроется в браузере автоматически.


СПОСОБ 2 (без сервера)
----------------------
Дважды кликните по файлу:

  vanguard.html

Работает, если в папке assets есть видео fonono.mp4.


ЧТО НУЖНО ПЕРЕДАТЬ ПОЛЬЗОВАТЕЛЮ
-------------------------------
Скопируйте или заархивируйте всю папку vanguard:

  vanguard.html      — главная страница
  OPEN.bat           — запуск в один клик
  assets/fonono.mp4  — фоновое видео (обязательно!)
  assets/            — остальные ресурсы

Node.js пользователям НЕ обязателен — сайт откроется и без него.


ПУБЛИКАЦИЯ В ИНТЕРНЕТЕ
-----------------------
Загрузите содержимое папки на любой хостинг:
  • Netlify, Vercel, GitHub Pages
  • обычный хостинг по FTP

Главный файл: vanguard.html


ТРЕБОВАНИЯ
----------
• Windows 10/11
• Любой браузер: Chrome, Edge, Firefox
• Интернет нужен только для шрифтов (сайт работает и без него)
